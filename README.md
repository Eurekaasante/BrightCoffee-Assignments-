# BrightCoffee-Assignments-
A portfolio of evidence to showcase the various SQL functions from assignment from Bright Learn Academy

Assignemnt 1. SQL Fundamentals Assignment: Data Retrieval & Filtering

Background & Objectives: This assignment introduces basic SQL syntax for querying a single table, such as an Employees or Products table. The primary objectives are to master SELECT, FROM, WHERE for filtering data, DISTINCT for removing duplicates, and ORDER BY for sorting results to produce organized reports.


Assignment 2. Aggregate Functions Assignment: Data Summarization
Background & Objectives: Moving beyond row-level data, this assignment focuses on summarizing datasets using aggregate functions to gain analytical insights. The objectives are to calculate statistics using SUM(), AVG(), COUNT(), MIN(), and MAX(), along with utilizing GROUP BY to categorize data and HAVING to filter groups.
Assignment 3. SQL Joins Assignment: Relational Data Retrieval 

Background & Objectives: This assignment addresses the challenge of retrieving data spread across multiple related tables, focusing on relational database principles. The main objectives are to understand and implement INNER JOIN, LEFT JOIN, and RIGHT JOIN to combine datasets based on common keys.
